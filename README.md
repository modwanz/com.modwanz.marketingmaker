## Features
* [x] **Fast video search**
* [x] **Dark and light themes**
* [x] **Create and import playlists**
* [x] **No need to log in to an account**
* [x] **Favorites and playlist shortcuts**
* [x] **Video playback in the background**
* [x] **Picture-in-Picture mode (floating video)**
